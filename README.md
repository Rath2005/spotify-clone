# 🎵 Spotify Clone

A full-stack Spotify Clone built using modern web technologies. This project allows users to listen to songs, create playlists, manage favorites, and enjoy a Spotify-like music streaming experience.

## 🚀 Features

- User Authentication (Register/Login)
- JWT-based Authorization
- Play Songs
- Create and Manage Playlists
- Recently Played Songs
- Favorite Songs
- Responsive User Interface
- Secure Backend APIs
- MongoDB Database Integration

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT (JSON Web Token)

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Rath2005/spotify-clone.git
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Configure Environment Variables

Create a `.env` file inside the backend folder and add:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
PORT=5000
```

### Run Backend

```bash
npm start
```

### Run Frontend

Open `frontend/index.html` using Live Server.
